## Channel
Color channel for the horizontal and vertical axes, so you can use `xChannel` and `yChannel` to specify which channel to use.
<How toUse="colors/color-area-channel-demo" />

## Space
The color space where the color area operates. The xChannel and yChannel gotta be in this color space. If not specified, it defaults to the color space of the color or defaultColor value. It has a type of `rgb` | `hsl` | `hsb`
<How toUse="colors/color-area-space-demo" />
