## Basic
This is the basic toaster component.
First, in your root app component, import the Toaster component.
```tsx
import { Toaster } from 'ui';
```
Then, add the Toaster component to your app.
```tsx
const App = () => {
  return (
    <Toaster />
  );
};
```
And next, just use the `toast()` function to show a toast.
