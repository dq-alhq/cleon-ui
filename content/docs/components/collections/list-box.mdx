## Basic

It can be customized to include features like multi-selection, filtering, and dynamic data updates. ListBoxes are commonly used in applications to present and manage lists of options or data entries.

## Rearrange

listBox lets you drag and drop stuff if you use the dragAndDropHooks prop with the useDragAndDrop hook. You can drop data on the whole list, individual items, add new items between old ones, or rearrange items.
