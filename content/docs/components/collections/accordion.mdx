## Basic

A vertically stacked set of interactive headings that each reveal a section of content.

<How className="w-full" toUse='collections/accordion-demo' />

## Installation

<Install cli='accordion' deps={['react-aria-components', 'framer-motion']} components={['accordion']} />

## Multiple Collapse

Set the initially expanded sections by providing an array of keys to the `defaultExpandedKeys` prop. These keys can be either numbers or strings.

<How className="w-full" toUse='collections/accordion-multiple-demo' />

## Hide Arrow Indicator

Hide the visual indicator (arrow key on the right) by using the `hideIndicator` prop.

<How className="w-full" toUse='collections/accordion-hide-indicator-demo' />

## Disabled

Disable specific sections by using the `disabledKeys` prop. For instance, keys 1, 2, and 4 are disabled in this example, making them non-interactive and unable to be expanded or collapsed.

<How className="w-full" toUse='collections/accordion-disabled-demo' />
